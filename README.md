##编写规范

- 缩进使用一个**TAB**，一个**TAB**代表`四个空格`
- 不同类型语法间使用一个空行，其余处不轻易使用空格和空行
- 一个完整的接口说明应包括接口描述、请求地址、请求方式、请求参数说明、请求参数示例、返回参数说明、返回示例、备注。[示例文档](example.md)
- 接口文件名应以序号开头，即章节号 + '.' + 小节号  + '.' + 空格
- 编写接口需手动添加目录，
